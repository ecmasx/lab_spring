# Cerinte

## Lab 1: Configurare Proiect și Clase de Bază

1. Configurare proiect:

   - Maven
   - Spring Boot 3.4.2
   - Java 17+

2. Crearea a 2-3 clase conform temei cu:

   - 5 tipuri de date
   - Constructor cu parametri
   - Constructor fără parametri
   - Setters și Getters
   - toString
   - equalsAndHashCode

3. Creare clasă main cu @SpringBootApplication

## Lab 2: Configurarea Lombok și Dependency Injection

1. Conectam Lombok în proiect

2. Adăugam următoarele anotări:

   - @AllArgsConstructor
   - @NoArgsConstructor
   - @Data
   - @RequiredArgsConstructor (unde este necesară injectarea prin constructor)

3. Eliminam metodele setter, getter, constructor, toString, equalsAndHashCode

4. Adăugam în clase referințe la alte clase pentru acces la date

5. Implementam dependency injection prin:

   - Constructor
   - Setter

6. Cream bean-uri prin:
   - Anotarea @Bean
   - Configurare XML
